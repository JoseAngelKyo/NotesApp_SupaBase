# 📝 Proyecto 01: NotesApp con Supabase

⭐ **5to Semestre - UPC** | **Julio 2026** 

---

## ¿Qué hace esta aplicación?
Aplicación web interactiva tipo Dashboard para **crear, organizar y eliminar notas en tiempo real**. Conectada a una base de datos en la nube con **Supabase (PostgreSQL)**, categorización visual por colores e íconos, e interfaz responsiva construida con **HTML, CSS y JavaScript Vanilla**.
